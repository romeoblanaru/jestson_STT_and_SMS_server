resulting in complete USB enumeration failure (kernel error -71) and
disappearance from USB bus. Physical unplug/replug required to recover.

================================================================================
KEY FINDINGS
================================================================================

1. PRIMARY ISSUE: Hardware/USB Communication Failure
   - Same kernel error -71 occurs with OR without AT+CGACT command
   - USB enumeration fails completely
   - Modem disappears from USB bus
   - Recovery requires physical unplug/replug

2. SECONDARY ISSUE: AT Commands Accelerate Crash
   - With rapid AT commands: crashes in 15-30 seconds
   - Without internet config: crashes in 90 seconds
   - Early I/O error during ATI command (reading modem details)
   
3. ABNORMAL BEHAVIOR: Port Renumbering
   - Expected: ttyUSB0-4
   - Observed: ttyUSB0,1,3,4,5 (ttyUSB2 missing, ttyUSB5 appeared)
   - Suggests USB re-enumeration instability

4. CONSISTENT CRASH PATTERN:
   - Always results in kernel error -71
   - Always requires physical reconnection
   - Occurs under multiple test conditions
   - Timing varies but crash is inevitable
